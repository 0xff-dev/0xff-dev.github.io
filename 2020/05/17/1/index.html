中序遍历情况下的节点的前驱/后继(二叉搜索树)

寻找节点的前驱分为三种情况



  1. 当node的左子树不是nil的时候，那么前驱就是直接找到左子树的最右边的节点即可。 图片里的13节点
  2. 当node的左子树为nil的时候如果node是父节点的右子树，那么前驱就是父节点 图片里的9节点
  3. 当node的左子树为nil的时候切是父节点的左子树，那么找到的节点就是：somenode.right.left….left = node 图片里的7节点

    在定义的树结构中没有parent指针的情况下，我们需要实现一个函数，定位node，nodeParent，以及第一个前驱节点.
    有父节点的情况直接利用父节点即可，

  4. 当node的右子树不为nil的情况，那么后继就是右子树的最左侧节点
  5. 当node的右子树为nil切node是父节点的左子树，那么后继就是父亲节点
  6. 当node的右子树为nil切node是父节点的右子树，那么需要找到的节点就是somenode.left.right…right = node
