python用tkinter实现贪吃蛇

  • python

   在看python的GUI编程， 发现tkinter上手难度还ok，就用他写一个贪吃蛇.

贪吃蛇

思路

主要的思路是把键盘事件与主界面的渲染分开, 🐍的操作就是上下左右，遇到墙，或者进自己的身体，游戏结束。对蛇单开一个线程
主界面就是canvas做矩形. 通过queue进行通信.

要掌握的知识

  1. Thread使用
  2. Queue使用
  3. 蛇的运动形成(遇到食物，直接在蛇尾添加， 而没有食物的情况，把去掉蛇尾，添加下一个位置即可)
  4. 矩形相交算法见函数is_rect_overlap.
